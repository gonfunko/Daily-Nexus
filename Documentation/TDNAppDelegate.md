#`TDNAppDelegate : UIResponder <UIApplicationDelegate>`#

TDNAppDelegate handles initial application setup and responding to application-wide state changes, specifically updating articles when the app resumes from the background after some time

##Public Interface##
`@property (strong, nonatomic) UIWindow *window` - The application's main window
